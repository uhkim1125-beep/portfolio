📂 데이터 구조
  • review.csv: 리뷰id, 구매id, 리뷰일자, 제품라인, 맛_종류, 구매옵션, 리뷰점수(1~5), 수량, VOC_대분류, VOC_태그
  • voc_master.csv: 대분류(9개 카테고리), 태그명, 참고키워드(콤마 구분)
  • 하나의 리뷰(리뷰id)에 복수의 VOC 태그가 멀티라벨로 부착됨
  • 리뷰 원문 대신 태그+대분류로 사전 분류된 구조
🏷️ 태깅 로직
  • voc_master 참고키워드를 리뷰 텍스트에 매칭 → 해당 태그명 부착
  • 한 리뷰에 여러 태그 적용 가능 (멀티라벨 분류)
  • 매칭 실패 시 기타/단순인사_분석불가로 처리
  • 구매옵션에서 맛/타입명 추출 (훈제, 갈릭, 저염, 마라, 커리, 허브갈릭, 큐브류 등)
  • 리뷰일자에서 연-월-일 분리 → 시계열 분석 활용
